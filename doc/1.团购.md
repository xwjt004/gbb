# 团购方法
后台套餐编辑页面已经有团购设置字段了。操作流程：                 
                                                                                     
  1. 管理后台 → 套系管理 → 套系列表 → 编辑某个套餐 
  2. 往下翻找到 团购设置 区域，填写：             
    - 成团人数（例如 3）
    - 团购价（例如 299）        
  3. 保存套餐   
  4. 小程序端 → 打开该套系详情页 → 就会看到 "🔥 拼团优惠" 横幅和底部 "拼团" 按钮
  5. 用户点击后进入开团页 → 发起拼团 → 分享给好友参团 。另外，管理后台的 营销管理 → 团购管理 可以查看所有团购活动的列表和状态。
  # 取消团购方法
  你现在可以去管理后台 营销管理 > 团购管理                                                  
  页面，找到进行中的团购，点击「取消」按钮将其标记为过期，然后就可以在微信小程序中重新开团测试了。
