# 微信服务号菜单

## 菜单结构

服务号底部三个一级菜单，每个菜单下包含若干子菜单项。

### 一级菜单：预约套系

| 子菜单 | 类型 | 说明 |
|---|---|---|
| 套系查看 | view | 跳转小程序套系列表页 |
| 团购活动 | view | 跳转小程序团购列表页 |

### 一级菜单：门店服务

| 子菜单 | 类型 | 说明 |
|---|---|---|
| 作品欣赏 | view | 跳转小程序作品展示页 |
| 导航到店 | view | 跳转小程序导航页 |
| 联系客服 | click | 点击回复客服电话 |

### 一级菜单：会员专区

| 子菜单 | 类型 | 说明 |
|---|---|---|
| 我的订单 | view | 跳转小程序订单列表页 |
| 会员中心 | view | 跳转小程序个人中心页 |
| 孕婴知识 | click | 点击推送孕婴知识内容 |

## 创建菜单

调用后端 API 创建菜单：

```bash
curl -X POST https://guaibaobao.cn/api/v1/wx-official-account/menu/create
```

后端使用微信公众平台 API `https://api.weixin.qq.com/cgi-bin/menu/create?access_token=ACCESS_TOKEN` 创建菜单。

## 关键词自动回复

用户在服务号聊天窗口发送文字消息时，系统自动匹配关键词回复。

| 关键词 | 回复内容 |
|---|---|
| 地址 | 店铺地址和导航信息 |
| 电话、客服 | 座机和手机号码，营业时间 |
| 价格 | 套餐价格档位介绍 |
| 套餐 | 引导点击菜单查看套系 |
| 预约 | 引导在线预约流程 |
| 新人 | 新人福利和优惠券 |
| 优惠 | 当前优惠活动介绍 |
| 帮助 | 所有关键词功能介绍 |
| 孕婴 | 孕婴知识目录：孕期/育儿/辅食/拍照技巧 |
| 孕期 | 孕期饮食建议和产检时间表 |
| 育儿 | 新生儿护理指南（喂养、睡眠、护理） |
| 辅食 | 辅食添加顺序和注意事项 |

无匹配关键词时回复默认引导信息，包含菜单功能说明和客服电话。

## 前置条件

创建菜单前需要：

1. AppSecret 已配置到服务器 `.env` 文件（`WX_OFFICIAL_ACCOUNT_APP_SECRET`）
2. 服务器配置已启用（微信公众平台后台 → 设置与开发 → 服务器配置 → 启用）
3. IP 白名单已添加服务器 IP `81.68.236.12`

## 跳转逻辑

view 类型菜单指向 `/jump?p=xxx` 轻量页面，该页面通过 URL Scheme 跳转小程序对应页面：

| 参数 p | 小程序页面 |
|---|---|
| packages | pages/packages/list/list |
| group-buys | pages/group-buy/list/list |
| portfolio | pages/works/works |
| store-nav | pages/store-nav/store-nav |
| orders | pages/order/list/list |
| profile | pages/profile/profile |
