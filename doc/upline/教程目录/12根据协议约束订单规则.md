# 订单业务规则与约束（基于协议）

---

## 一、核心规则：仅已支付订单在后台生效

微信小程序下单后，只有完成支付的订单才会进入管理后台的订单管理列表。

| 订单来源 | 支付状态 | 后台显示位置 | 档期占用 |
|---------|---------|:-----------:|:--------:|
| 小程序 | 未支付 | 潜在订单列表（单独入口） | 不占用 |
| 小程序 | 已付定金 / 已全款 | 订单管理列表 | 占用 |
| 管理后台 | 不限 | 订单管理列表 | 占用 |

**要点**：
- 未支付订单为"草稿"，商家无需处理，无需确认
- 档期在客户实际支付后才锁定
- 支付到账后订单自动进入待处理队列
- 未支付订单超过90天自动清理释放

---

## 二、订单生命周期

```
客户小程序下单
    │
    ├── 未支付（潜在订单）→ 超时90天自动取消
    │
    ├── 支付订金 → 待确认 → 商家确认 → 已确认
    │                    └→ 客户付尾款后商家确认
    │
    └── 全款支付 → 待确认 → 商家确认 → 已确认

已确认（可取消） → 拍摄中 → 已完成（终态）
                                 （不可取消）
```

## 三、订单操作规则

### 3.1 确认订单（商家操作）

- **条件**：订单状态为"待确认"，且客户已支付（定金或全款）
- **效果**：订单变为"已确认"，安排拍摄档期
- **注意**：未支付订单不可确认

### 3.2 取消订单

#### 客户取消（微信小程序）

仅"待确认"和"已确认"状态的订单可取消，"已完成"不可取消。

| 取消时间 | 已付定金退款规则 | 已全款退款规则 |
|---------|:---------------:|:-------------:|
| 商家确认前 | 全额退款 | 全额退款 |
| 确认后 → 拍摄前7天 | 退扣除定金后剩余部分 | 扣30%，退70% |
| 拍摄前7天 → 拍摄前24小时 | 定金不退 | 扣50%，退50% |
| 拍摄前24小时内 | 定金不退 | 不退 |
| 爽约未到店 | 定金不退 | 扣30%，退70% |

#### 商家取消（管理后台）

| 原因 | 扣款规则 |
|-----|---------|
| 商家原因（档期冲突、摄影师请假等） | 全额退款 |
| 客户违约（超时未付尾款等） | 按合同约定扣款后退还 |
| 已完成订单 | 不可取消，走退款流程 |

### 3.3 删除订单

- 仅"已取消"状态的订单可以删除
- 小程序端：软删除（数据保留但不可见）
- 管理后台：硬删除

## 四、支付与退款

### 4.1 支付模式

| 模式 | 说明 |
|------|------|
| 全款在线支付 | 下单时一次性付清（默认） |
| 定金在线支付 | 先付订金，尾款后续补齐 |
| 线下支付（全款/定金） | 到店付现金或刷卡 |

### 4.2 订金规则

- **订金由商户在创建套系时设定**，支持两种方式：
  - **固定金额**：如 ¥500
  - **按百分比**：如总价的 30%
- 订金设为零即为"全款支付"模式
- 订金支付后订单进入"待确认"状态
- 尾款 = 总价 − 订金金额
- 尾款补齐后订单变为"已全款"

### 4.3 退款流程

```
客户申请退款 → 商家审批 → 执行退款 → 退款完成
```

| 退款类型 | 说明 |
|---------|------|
| 全额退款 | 全部退还，订单自动取消 |
| 部分退款 | 退还部分金额，订单继续有效 |
| 仅退定金 | 仅退还已付定金部分 |

| 退款方式 | 说明 |
|---------|------|
| 原路退回 | 通过微信支付原路退还（需开通退款API） |
| 线下退款 | 现金或银行转账 |

### 4.4 退款扣款标准

| 原因 | 客户承担 |
|-----|:--------:|
| 客户个人原因 | 按取消时间节点阶梯扣款 |
| 商家原因 | 0%（全额退） |
| 双方协商 | 按协商比例 |
| 服务质量问题 | 0%（全额退），可额外赔偿 |

| 操作 | 微信小程序 | 管理后台 |
|-----|:---------:|:--------:|
| 查看订单 | ✅ | ✅ |
| 取消订单 | ✅（待确认/已确认） | ✅（已完成除外） |
| 删除订单 | ✅（仅已取消） | ✅（仅已取消） |
| 确认订单 | ❌ | ✅ |
| 完成订单 | ❌ | ✅ |
| 退款 | ❌（联系客服） | ✅ |
| 支付 | ✅（微信支付） | ✅（线下收款） |
