## 一、系统概述

乖宝宝儿童影楼管理系统是一套面向儿童摄影门店的 **B 端管理平台**，涵盖订单、财务、产品、客户、营销、供应链等全链路管理功能。

- 访问地址：`https://guaibaobao.cn/admin/`
- 技术栈：React 18 + Ant Design 5 + NestJS 11 + PostgreSQL 17
- 权限体系：基于角色的访问控制（RBAC），支持管理员/普通员工两种角色

## 三、详细功能说明

### 模块 1：仪表盘

**路径**：侧边栏 → 仪表盘

#### 概览页
- **统计卡片**：当日订单数、收入金额、新增客户数、待处理事项
- **快捷入口**：新建订单、查看日程、客户查询
- **待办提醒**：待审核退款、待处理客诉、过期未成团团购

#### 数据看板
- 展示更详细的业务趋势图表（订单趋势、收入走势、套餐销量排行）
- 支持按时间范围筛选

---

### 模块 2：套系管理

**路径**：侧边栏 → 产品中心 → 套系列表 / 套系分类

#### 套系列表
- **展示**：套系缩略图、名称、原价、团购价、状态、排序号
- **操作**：新增、编辑、删除、查看详情
- **编辑页功能**：
  - 基本信息：名称、原价、描述、缩略图、详情图片
  - 服务内容：关联服务项目（可多选并设置数量）
  - 商品内容：关联商品（可多选并设置数量）
  - 团购设置：成团人数、团购价、团购说明
  - 阶梯团购价：添加/编辑/删除阶梯（如 2人=¥250, 5人=¥200）
  - 团购海报设置：海报标题、描述、背景、宣传照片
  - 季节性价格：按日期设置不同价格

#### 套系分类
- 管理套系所属分类（如：新生儿、百天、周岁、亲子等）
- 支持排序、启用/禁用

---

### 模块 3：商品管理

**路径**：侧边栏 → 产品中心 → 商品列表 / 商品分类

#### 商品列表
- **展示**：商品缩略图、名称、价格、库存、销量、状态
- **操作**：新增、编辑、删除
- **编辑页**：商品名称、价格、库存、描述、图片、规格

#### 商品分类
- 管理商品所属分类
- 支持排序

---

### 模块 4：服务项目

**路径**：侧边栏 → 产品中心 → 服务项目

- 管理拍摄相关的服务项目（如：精修张数、服装套数、入册张数等）
- 项目可在套系编辑中关联引用

---

### 模块 5：DIY套系

**路径**：侧边栏 → 产品中心 → DIY套系

#### DIY套系列表
- 展示所有用户自定义组合的套系
- 支持编辑、删除

#### DIY套系构建器
- 可视化组合套餐：选择基础套系 → 叠加服务项目/商品
- 自动计算价格

#### DIY折扣规则
- 设置 DIY 套系的折扣策略（如满减、百分比折扣）

---

### 模块 6：订单管理

**路径**：侧边栏 → 订单管理

#### 订单列表
- **展示**：订单号、套餐/商品信息、客户信息、金额、支付状态、拍摄状态、时间
- **筛选**：按状态（待付款/已付款/已完成/已取消）、日期范围、关键词搜索
- **操作**：
  - 查看详情：订单完整信息、支付记录、拍摄安排
  - 编辑订单：修改订单信息（部分字段）
  - 取消订单
- **导出**：支持导出订单数据为 Excel

#### 团购订单
- **展示**：团购编号、套餐/商品、团长信息、团购价、入团人数、状态
- **筛选**：按状态（进行中/已成团/已失败）、日期范围
- **操作**：详情、编辑（成团人数/参团上限/过期时间）、撤团、恢复、删团
- **导出**：支持导出团购数据为 Excel

---

### 模块 7：客户管理

**路径**：侧边栏 → 客户管理

#### 客户信息
- 展示所有微信授权用户
- 字段：昵称、头像、手机号、性别、注册时间、消费次数、积分
- 操作：查看详情（订单历史、积分明细、优惠券）、编辑备注

#### 会员等级
- 设置会员等级体系（如普通会员、银卡、金卡、钻石）
- 配置各等级的升级条件、权益（折扣率、积分倍数等）

#### 客诉管理
- 记录和处理客户投诉
- 状态跟踪：待处理 → 处理中 → 已解决
- 支持关联订单

---

### 模块 8：拍摄管理

**路径**：侧边栏 → 拍摄管理

#### 时间槽管理
- 管理每日可预约的时间段
- 设置：日期、时段（如 9:00-11:00）、可用数量
- 支持批量生成时间槽

#### 拍摄日程
- 以日历视图展示每日拍摄安排
- 展示每个时间段的预约客户、套餐信息
- 支持拖拽调整

---

### 模块 9：财务管理

**路径**：侧边栏 → 财务管理

#### 支付列表
- 展示所有支付记录
- 字段：支付单号、订单号、支付金额、支付方式（微信支付）、支付状态、时间
- 支持按状态、日期筛选

#### 可疑支付检测
- 自动检测异常支付（金额不匹配、状态不一致）
- 风险等级：高/中/低
- 支持标记处理

#### 对账管理
- 每日自动对账（凌晨 3:00）
- 检测订单金额与实际收款差异
- 展示对账报告：差异明细、处理状态

#### 退款管理
- **退款审批**：待审核退款列表，支持通过/拒绝
- **退款记录**：已完成退款的历史记录

---

### 模块 10：营销管理

**路径**：侧边栏 → 营销管理

#### 优惠券管理
- 创建优惠券模板：类型（折扣/满减）、面值、适用范围、有效期
- 发放优惠券：手动发放、批量发放
- 支持成团后自动发放优惠券（团购场景）

#### 积分配置
- 设置积分规则：消费 1 元 = N 积分、积分抵现比例
- 会员等级积分倍数

#### 积分明细
- 查看所有客户的积分变动记录
- 筛选：按客户、时间范围

#### 客户分群
- 基于标签/消费行为创建客户分群
- 支持分群：高价值客户、沉睡客户、新客户等
- 分群可用于精准营销活动

#### 营销活动
- 创建和管理营销活动
- 活动类型：满减、折扣、赠品等
- 活动效果追踪：参与人数、转化率

#### 数据分析
- ****
  > 数据分析模块涵盖经营报表、趋势分析、品类分析等

---

### 模块 11：供应链管理

**路径**：侧边栏 → 供应链管理

#### 供应商列表
- 管理供应商信息：名称、联系人、电话、地址
- 支持黑名单管理

#### 采购订单
- 创建采购订单：选择供应商、商品、数量、价格
- 采购审批流程
- 订单状态跟踪：待审批 → 已审批 → 已发货 → 已完成

#### 在途商品
- 展示已发货但未入库的商品
- 跟踪物流状态

#### 入库管理
- 采购到货后的入库操作
- 库存自动更新

---

### 模块 12：库存智能

**路径**：侧边栏 → 供应链管理 → 库存智能

#### 销量预测
- 基于历史数据预测未来销量
- 辅助采购决策

#### 安全库存
- 设置各商品的安全库存阈值
- 低于阈值时触发预警

#### 补货建议
- 自动生成补货建议（基于销量预测 + 安全库存）
- 支持一键转为采购订单

#### 呆滞预警
- 识别长期未动销商品
- 支持设置呆滞判定标准（如 90 天未销售）

#### 周转分析
- 库存周转率计算和趋势分析
- 按商品/分类维度展示

---

### 模块 13：系统管理

**路径**：侧边栏 → 系统管理

#### 员工列表
- 管理后台登录账号
- 字段：昵称、手机号、角色、状态
- 操作：新增、编辑、启用/禁用、重置密码

#### 角色管理
- 创建和管理角色
- 权限配置：按模块/操作细粒度授权（如"订单管理：查看"、"订单管理：编辑"）

#### 系统设置
- 通用配置项
- 修改密码

#### 店铺信息
- 店铺名称、电话、地址、营业时间
- 展示在微信小程序端

#### 打印设置
- 配置打印格式（如小票、单据）

#### 自动化规则
- 设置业务自动处理规则（如预约提醒、自动对账等）

#### 操作日志
- 记录所有用户的关键操作（增/删/改）
- 字段：操作人、操作类型、操作对象、IP、时间

#### 系统状态
- 服务器健康状况：CPU/内存/磁盘
- 服务运行状态

#### 数据库备份/恢复
- 手动触发数据库备份
- 查看已有备份文件
- 支持备份恢复

#### 更改主题
- 切换系统主题色（支持浅色/深色主题）

## 四、常见操作流程

### 4.1 新建订单
1. 侧边栏 → 订单管理 → 点击"新建订单"
2. 选择客户（微信用户）→ 选择套餐/商品 → 确认金额
3. 如有团购活动，自动关联团购订单
4. 提交后生成订单，客户可在小程序端查看

### 4.2 处理退款
1. 侧边栏 → 财务管理 → 退款审批
2. 查看退款申请详情 → 确认订单和金额
3. 点击"通过"（自动发起微信支付退款）或"拒绝"

### 4.3 管理团购
1. 侧边栏 → 订单管理 → 团购订单
2. 查看进行中的团购 → 可手动撤团（标记失败）
3. 查看已失败的团购 → 可恢复（重新激活）
4. 阶梯价格设置在套系编辑页中管理

### 4.4 数据导出
1. 侧边栏 → 工具 → 导出中心
2. 选择导出类型：订单/员工/财务/全部
3. 设置筛选条件 → 点击导出 → 下载 Excel

## 五、注意事项

1. **权限说明**：管理员拥有全部权限，普通员工只能访问被授权的模块
2. **数据一致性**：财务相关的操作（退款、对账）不可逆，操作前请确认
3. **团购特殊规则**：
   - 撤团后活动变为 FAILED，可恢复；删团为硬删除不可恢复
   - 过期团购自动标记 FAILED 并自动退款
4. **备份建议**：建议定期手动备份数据库，尤其在重大操作前
