  3. 重装系统                                                                                                                                                                                                                        
  1. 关机：选中实例 → 更多 → 实例状态 → 关机
  2. 重装：确认关机后 → 更多 → 重置系统
  3. 选择镜像：
    - 镜像类型：公共镜像
    - 操作系统：Ubuntu Server 22.04 LTS 64位
    - （保持和之前一致的版本）
  4. 系统盘：保持默认（不调整大小）
  5. 设置密码：设置新密码（不要用旧密码 Ly-760606@，换一个更复杂的）
  6. 安全组确认：重装后安全组规则会保留，但要确保以下端口已开放：
    - 80（HTTP）
    - 443（HTTPS）
    - 22（SSH-仅从你本地IP访问）

  ⚠️  删除不必要的端口：
    - ~~5432~~ ❌（PostgreSQL - 被攻击原因！）
    - ~~6379~~ ❌（Redis）
    - ~~3000~~ ❌（后端，通过Nginx反向代理访问）

  4. 重装后检查
