## 一、小程序概述

乖宝宝儿童影楼微信小程序是面向**消费者**的移动端应用，提供在线预约拍摄、选购商品、参与团购、查看订单等功能。

- 入口：微信搜索"乖宝宝儿童摄影"小程序
- 技术栈：原生微信小程序 + Vant Weapp UI
- 底部导航：预约 | 商品 | 购物车 | 我的

## 三、详细功能说明

### 模块 1：套系浏览

**入口**：底部导航"预约"Tab / 首页

#### 套系列表页
- 以卡片网格形式展示所有可用套系（套餐）
- 每个卡片显示：缩略图、名称、原价、团购价（如有）
- 支持按分类筛选（如：新生儿、百天、周岁等）
- 支持关键词搜索

#### 套系详情页
- **套餐信息**：名称、价格、详细描述、多张展示图片
- **服务内容**：列出包含的服务项目（精修张数、服装套数等）
- **商品内容**：列出包含的商品
- **团购信息**：如果该套系支持团购，显示团购价格和阶梯价格表
- **操作按钮**：
  - "立即预约" → 进入预约流程
  - "拼团优惠" → 查看团购详情/开团

---

### 模块 2：商品浏览

**入口**：底部导航"商品"Tab

#### 商品列表页
- 展示所有在售商品（如：相框、摆台、纪念品等）
- 支持按分类筛选
- 支持关键词搜索

#### 商品详情页
- **商品信息**：名称、价格、库存状态、描述、多张图片
- **操作**：
  - "加入购物车" → 选择数量后加入
  - "立即购买" → 直接进入下单流程

---

### 模块 3：购物车

**入口**：底部导航"购物车"Tab

- 展示已加入购物车的商品列表
- **操作**：
  - 修改数量（步进器）
  - 删除商品（左滑或勾选删除）
  - 选择商品 → 点击"结算" → 进入订单确认
- 购物车数据跨 session 持久化

---

### 模块 4：预约拍摄

**入口**：套系详情页 → 点击"立即预约"

预约流程为四步：

#### 步骤一：选择拍摄日期
- 日历控件选择日期
- 标记可预约和已约满的日期

#### 步骤二：选择拍摄时段
- 展示所选日期下的可用时间段
- 每个时段显示剩余可预约数量

#### 步骤三：填写联系信息
- 联系人姓名、手机号
- 备注信息（可选）

#### 步骤四：确认订单
- 确认套餐信息、预约时间、金额
- 如有优惠券，可选择使用
- 确认后进入支付

---

### 模块 5：订单管理

**入口**：我的 → 我的订单

#### 订单列表页
- 按状态分类展示：全部 / 待付款 / 进行中 / 已完成 / 已取消
- 每个订单显示：套餐/商品缩略图、名称、金额、状态、时间

#### 订单详情页
- **订单信息**：订单号、套餐/商品详情、金额、支付方式、下单时间
- **预约信息**：拍摄日期、时段、门店地址
- **状态流转**：
  - 待付款 → 去支付
  - 已支付/进行中 → 查看预约详情
  - 已完成 → 查看相册
  - 已取消 → 重新购买

---

### 模块 6：团购

**入口**：套系详情页 → "拼团优惠" / 分享链接 / 我的 → 我的团购

#### 开团流程
1. 进入套系详情 → 点击"拼团优惠"
2. 查看团购价格和阶梯优惠
3. 设置成团人数（可调整）
4. 点击"立即开团" → 开团成功成为团长
5. 自动跳转团购详情页

#### 团购详情页
- **团购信息**：套餐名称、团购价、当前参与人数、成团人数、倒计时
- **参团者列表**：显示所有参团者头像、昵称
- **阶梯价格展示**：参团人数对应的阶梯价格表
- **操作**：
  - "立即参团" → 加入该团购
  - "分享给好友" → 通过微信分享
  - "生成海报" → 生成带二维码的团购海报
  - "分享到朋友圈" → 保存图片后发布
- **团长特权**：团长可看到"分享海报"入口

#### 参团流程
1. 通过分享链接/二维码进入团购详情
2. 点击"立即参团"
3. 自动匹配当前人数的阶梯价格
4. 进入预约流程（选日期→时段→信息→确认）
5. 完成支付 → 参团成功

#### 我的团购列表
- 展示所有参与和发起的团购
- 每个团购显示：套餐名称、状态（进行中/已成团/已失败）、人数进度

#### 退团
- 仅限进行中状态的团购
- 仅限非团长参团者操作
- 已支付的订单需联系客服处理退款

---

### 模块 7：支付

**入口**：订单确认页 → 点击"立即支付"

#### 支付流程
1. 确认订单信息 → 点击支付
2. 调起微信支付（统一下单 API）
3. 输入支付密码 → 支付成功
4. 跳转支付结果页

#### 支付结果页
- 支付成功：显示成功状态、订单信息、"查看订单"按钮
- 支付失败：显示失败原因、"重新支付"按钮

#### 支付协议
- 首次支付前需签署《微信支付委托代扣协议》
- 在支付页面的协议区域可查看和签署

---

### 模块 8：优惠券

**入口**：我的 → 优惠券

#### 优惠券列表

- 每张优惠券显示：面值/折扣、适用范围、有效期

#### 使用优惠券
- 在订单确认页可选择可用优惠券
- 同一订单仅可使用一张优惠券
- 团购成团后自动发放 9 折优惠券（限该套餐）

## 四、典型用户流程

### 4.1 用户首次使用流程
```
打开小程序 → 微信授权登录 → 浏览套系
→ 选择套系 → 预约拍摄 → 选日期 → 选时段
→ 填写信息 → 确认订单 → 微信支付
→ 到店拍摄 → 线上查看相册
```

### 4.2 团购裂变流程
```
用户 A 浏览套系 → 看到团购优惠 → 发起开团
→ 分享团购链接给好友 B、C
→ B、C 点击链接参团 → 完成各自预约和支付
→ 达到成团人数 → 团购成功 → 自动发放优惠券
```

### 4.3 商品购买流程
```
浏览商品 Tab → 选择商品 → 加入购物车
→ 进入购物车 → 选择商品 → 结算
→ 确认收货地址 → 微信支付 → 等待发货
```

## 五、常见问题

### Q1: 如何支付？
在订单确认页点击"立即支付"，系统自动调起微信支付，输入密码即可完成。

### Q2: 如何查看已预约的拍摄？
进入"我的" → "我的订单"，点击对应订单可查看预约详情（日期、时段、门店地址）。

### Q3: 如何参与团购？
通过好友分享的团购链接进入，或浏览套系时点击"拼团优惠"，选择已有团购参团或自己开团。

### Q4: 团购有效期多久？
团购有效期为 48 小时，过期未达人数自动失败并退款。

### Q5: 团购失败怎么办？
系统会自动为已支付的参团者退款，退款原路返回微信支付账户。

### Q6: 如何分享给好友？
在团购详情页点击右上角菜单 → "转发给好友"，或点击"生成海报"保存图片后分享。

### Q7: 优惠券如何使用？
下单时在订单确认页可选择适用优惠券自动抵扣。

### Q8: 如何联系商家？
在"我的" → "关于"中查看商家联系电话和地址。

## 六、注意事项

1. **网络要求**：小程序依赖微信网络环境，弱网状态下可能导致支付超时
2. **授权说明**：首次使用需授权微信登录，拒绝授权将无法使用完整功能
3. **团购规则**：
   - 同一用户同时只能有一个进行中的团购（作为团长）
   - 每人只能参与同一团购一次
   - 达到参团上限后无法继续参团
4. **退款说明**：退团后已支付的订单不自动退款，需联系客服处理
5. **版本更新**：小程序更新需要微信审核，新功能上线有审核周期
