# 乖宝宝微信小程序 — 上传图片规格说明

## 设计基准

| 项目 | 值 |
|------|-----|
| 小程序屏幕宽度 | 750rpx |
| 设计稿宽度 | 750px |
| 推荐分辨率 | **72dpi**（Web标准） |
| 色彩模式 | **sRGB** |
| 图片格式 | **JPEG**（照片类）、**PNG**（透明背景） |
| 单张大小 | **不超过 2MB** |

---

## 一、商品封面图 / 套系封面图

> 在商品列表、套系列表中展示的主图，卡片内为固定比例裁剪。

| 规格项 | 要求 |
|--------|------|
| 推荐尺寸 | **750 × 750 px**（1:1 正方形） |
| 最小尺寸 | 500 × 500 px |
| 最大尺寸 | 1200 × 1200 px |
| 宽高比 | **1:1**（代码中 `mode: "aspectFill"` 居中裁剪） |
| 文件格式 | JPEG |
| 文件大小 | ≤ 500 KB |
| 说明 | 图片会以正方形卡片展示，建议主体居中，避免重要内容靠近边缘 |

> **备选比例（若不想用正方形）：**
> 也可使用 **750 × 500 px**（3:2 横图），但卡片区域会上下留白。

---

## 二、详情展示图

> 商品详情页 / 套系详情页中的展示图片，可多张，会以轮播或列表方式展示。

| 规格项 | 要求 |
|--------|------|
| 推荐尺寸 | **750 × 1000 px** |
| 宽高比 | **3:4**（竖图，适合手机浏览） |
| 最大宽度 | 750 px（超过会自动缩放） |
| 最大高度 | 不限，但建议 ≤ 1500 px |
| 文件格式 | JPEG |
| 文件大小 | 每张 ≤ 500 KB |

---

## 三、轮播图（Banner）

> 首页或列表页顶部的大图轮播。

| 规格项 | 要求 |
|--------|------|
| 推荐尺寸 | **750 × 380 px** |
| 宽高比 | **约 2:1** |
| 最大尺寸 | 1125 × 570 px（1.5倍以适配高分屏） |
| 文件格式 | JPEG |
| 文件大小 | ≤ 500 KB |
| 安全区域 | 左右各留 **30px** 内边距避免被裁切（文字不要放在边缘） |

> 说明：轮播图容器高度在不同页面略有差异（350~420rpx），750×380 是通用安全值。

## 六、图片上传注意事项

1. **命名规范**（建议）：
   - 商品：`product-{名称}-{序号}.jpg`
   - 套系：`package-{名称}-{序号}.jpg`
   - 轮播：`banner-{名称}-{序号}.jpg`
   - 文件名不要含中文和空格，用英文或拼音 + 连字符

2. **压缩建议**：
   - 使用 TinyPNG (tinypng.com) 或类似工具压缩 JPEG/PNG
   - 压缩后肉眼无明显画质损失即可（通常可压缩 50%~70%）
   - 工具推荐：TinyPNG、Squoosh、Photoshop「导出为 Web 格式」

3. **色彩**：
   - 摄影类照片色彩空间使用 **sRGB**，不要使用 Adobe RGB 或 ProPhoto RGB
   - sRGB 在微信内的显示最准确

4. **禁止内容**：
   - 不得包含二维码（微信限制）
   - 不得包含联系方式（电话/微信等）
   - 不得包含其他平台水印
