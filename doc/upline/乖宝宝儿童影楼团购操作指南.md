# 乖宝宝儿童影楼团购操作指南（管理后台）
管理后台操作：团购列表、筛选导出、创建编辑、阶梯价管理、套餐设置、数据统计
## 一、访问入口

管理后台 URL：`https://guaibaobao.cn/admin/`

团购管理路径：左侧导航栏 → **订单管理** → **团购订单**

---

## 二、团购订单列表

### 2.1 统计卡片

页面顶部展示四项统计数据：

| 指标 | 说明 |
|------|------|
| 总团购数 | 系统中所有团购活动总数 |
| 进行中 | 当前状态为 ACTIVE 的活动数量 |
| 已成团 | 状态为 SUCCESS 的活动数量 |
| 已失败 | 状态为 FAILED 的活动数量 |

### 2.2 筛选与搜索

| 筛选条件 | 说明 |
|---------|------|
| 搜索框 | 按套餐名称关键词搜索 |
| 日期范围 | 按创建时间筛选 |
| 状态筛选 | 全部 / 进行中 / 已成团 / 已失败 |

### 2.3 表格列说明

| 列名 | 说明 |
|------|------|
| 团购编号 | 显示 UUID 前 8 位 |
| 商品信息 | 商品缩略图、名称、类型标签（套餐/商品）、团购价与原价对比 |
| 团长 | 开团人昵称和手机号 |
| 团购价 | 显示团购优惠价格 |
| 入团人数 | 当前人数 / 成团人数（上限人数） |
| 已付款 | 已支付订单数 |
| 未付款 | 未支付订单数 |
| 状态 | 进行中（蓝色）/ 已成团（绿色）/ 已失败（红色） |
| 创建时间 | 活动创建时间 |
| 过期时间 | 活动过期时间（过期未成团自动标记失败） |

### 2.4 操作按钮

| 操作 | 说明 |
|------|------|
| 详情 | 查看活动完整信息 |
| 编辑 | 修改成团人数、参团上限、过期时间 |
| 撤团 | 将进行中的活动标记为失败（仅 ACTIVE 状态可用） |
| 恢复 | 将已失败的活动恢复为进行中（仅 FAILED 状态可用） |
| 删团 | 硬删除活动及所有参团记录（不可恢复） |

### 2.5 导出

点击页面右上角"导出"按钮，可导出当前列表数据为 Excel 文件，包含：

- 团购编号、套餐名称、原价、团购价
- 团长昵称、团长手机
- 入团人数、成团人数、参团上限
- 已付款、未付款
- 状态、创建时间、过期时间

## 三、创建团购

### 3.1 手动创建

1. 点击页面右上角"**新建团购**"按钮
2. 在弹出的对话框中填写：
   - **选择套餐**：搜索并选择要开团的套餐
   - **开团人**：搜索并选择团长（微信用户）
   - **成团人数**：达到该人数则团购成功（默认 3 人）
   - **参团人数上限**（选填）：限制最多参团人数，不填则不限制
3. 点击"**创建**"按钮

### 3.2 编辑团购

1. 点击目标团购的"**编辑**"按钮
2. 在弹窗中可修改：
   - **成团人数**
   - **参团上限**
   - **过期时间**
3. 团购价仅在套餐编辑中修改（详情展示区只读）

## 四、阶梯价格管理

### 4.1 进入阶梯价格设置

路径：左侧导航栏 → **套餐管理** → **套餐列表** → 点击目标套餐的"**编辑**"按钮 → 在编辑页面中找到"**阶梯团购价**"区域

### 4.2 添加阶梯价

1. 在"阶梯团购价"区域点击"**添加阶梯**"按钮
2. 填写：
   - **成团人数**：达到该人数时适用此价格（最小 2 人）
   - **阶梯价格**：该阶梯对应的优惠价格
3. 点击"**确定**"

> 注意：同一人数不能重复设置阶梯价，系统会提示"该人数阶梯已存在"

### 4.3 编辑/删除阶梯价

- 点击阶梯价条目旁的"**编辑**"图标修改价格或人数
- 点击"**删除**"图标移除该阶梯价

### 4.4 价格匹配规则

阶梯价格按人数从低到高排列，参团时自动匹配当前人数适用的价格：

| 阶梯设置 | 参团 2 人 | 参团 5 人 | 参团 8 人 |
|---------|----------|----------|----------|
| 2人=¥250 | ¥250 | ¥250 | ¥250 |
| 5人=¥200 | ¥250 | ¥200 | ¥200 |
| 10人=¥180 | ¥250 | ¥200 | ¥180 |

> 取所有满足条件（参团人数 ≥ 阶梯人数）中人数最高的阶梯价格。

## 五、套餐团购设置

### 5.1 团购基础设置

在套餐编辑页的"团购设置"区域：

| 字段 | 说明 |
|------|------|
| 成团人数 | 该套餐的默认成团人数（小程序端开团时自动填充） |
| 团购价 | 前端展示的团购优惠价（仅展示，不参与订单结算） |
| 团购说明 | 团购活动描述文字 |

### 5.2 团购海报设置

在套餐编辑页的"团购海报设置"区域：

| 字段 | 说明 |
|------|------|
| 海报标题 | 海报上的标题文字（最多 200 字） |
| 海报描述 | 海报上的描述文字（最多 1000 字） |
| 海报背景 | 背景色代码或背景图片 URL |
| 宣传照片 | 上传宣传图片（支持预览） |

## 七、注意事项

1. **撤团操作不可逆**：撤团后活动变为 FAILED 状态，可恢复但需手动操作
2. **删除操作不可恢复**：删团会同时清除所有参团记录和订单关联
3. **过期自动处理**：活动过期（创建后 48 小时）后自动标记为 FAILED，并自动退款
4. **优惠券自动发放**：成团后系统自动为所有参团者发放优惠券（无需手动操作）
5. **商品团购**：如为商品（非套餐）团购，成团后不发放优惠券
