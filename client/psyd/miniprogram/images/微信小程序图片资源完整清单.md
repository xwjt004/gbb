## 🔄 图片替换流程

### 替换单个图片

```powershell
# 1. 准备新图片
# 2. 替换对应文件
cp 新图片路径 images/原图片名.png

# 示例：替换商品Tab图标
cp 设计稿/商品图标.png images/tab-product.png
cp 设计稿/商品图标-选中.png images/tab-product-active.png
```

### 批量替换

```powershell
# 将所有设计好的图片放到一个目录
# 然后批量复制到对应位置
cd d:\vscode\GBB\client\psyd\miniprogram
cp 设计稿/tab-*.png images/
cp 设计稿/empty-*.png images/
cp 设计稿/icon-*.png images/
```

## 📝 注意事项

### 1. 当前使用占位符

所有图片都是从 `placeholder.png` 复制而来的占位符。  
建议准备正式设计的图片后进行替换。

### 2. 文件格式

- **图标/Logo**: PNG格式（支持透明背景）
- **Banner/照片**: JPG格式（文件更小）

### 3. 图片优化

替换正式图片时，请确保：
- 图片已压缩优化
- 文件大小控制在合理范围
- 清晰度满足显示需求

### 4. 测试验证

替换图片后，请在微信开发者工具中：
1. 清除缓存
2. 重新编译
3. 检查所有页面显示
