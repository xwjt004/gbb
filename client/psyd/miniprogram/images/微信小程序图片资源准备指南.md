## 📝 图片优化建议

### 1. 压缩图片

**在线工具**:
- TinyPNG: https://tinypng.com/
- Squoosh: https://squoosh.app/

**目标**: 
- 图片大小 < 200KB
- 保持清晰度

### 2. 使用WebP格式

微信小程序支持WebP格式，文件更小。

### 3. 懒加载

在列表页面使用图片懒加载，提升性能。

## 🔧 图片替换步骤

### 替换Tab图标

```bash
# 将设计好的图标放到images目录
cp 新图标/tab-product.png images/
cp 新图标/tab-product-active.png images/
# ... 其他图标
```

### 替换Banner图片

```bash
# 将设计好的Banner放到banners目录
cp 新图片/banner1.jpg banners/
cp 新图片/banner2.jpg banners/
```

### 替换Logo

```bash
# 将Logo放到images目录
cp 新Logo/logo.png images/
```

然后修改 `pages/welcome/welcome.wxml`:
```xml
<!-- 将emoji替换为图片 -->
<image src="/images/logo.png" class="brand-logo" mode="aspectFit" />
```
